C++ Fundamentals: The Sorting TrilogyA clean, efficient implementation of the three most common $O(n^2)$ sorting algorithms in C++. This repository is designed for educational purposes, demonstrating how different logic structures can achieve the same goal of ordering data.📚 Algorithms Overview1. Selection Sort 🔍Selection sort works by repeatedly finding the minimum element from the unsorted part of the array and putting it at the beginning.How it works: It maintains two subarrays: one already sorted and one remaining to be sorted. In every iteration, the minimum element from the unsorted subarray is picked and moved to the sorted subarray.Best/Worst Case: $O(n^2)$ (It always scans the entire remaining list).2. Bubble Sort (Optimized) 🫧Bubble Sort works by repeatedly swapping adjacent elements if they are in the wrong order.How it works: Small elements "bubble" to the top. This version includes an early exit optimization: if the inner loop finishes without a single swap, the algorithm stops because the array is already sorted.Best Case: $O(n)$ (when already sorted).Worst Case: $O(n^2)$.3. Insertion Sort 🃏Insertion sort works similarly to the way you sort playing cards in your hands.How it works: The array is virtually split into a sorted and an unsorted part. Values from the unsorted part are picked and placed at the correct position in the sorted part by shifting larger elements out of the way.Best Case: $O(n)$.Worst Case: $O(n^2)$.📈 Complexity Comparison TableAlgorithmTime Complexity (Avg)Time Complexity (Best)Space ComplexityStable?Selection Sort$O(n^2)$$O(n^2)$$O(1)$NoBubble Sort$O(n^2)$$O(n)$$O(1)$YesInsertion Sort$O(n^2)$$O(n)$$O(1)$Yes🛠️ Requirements & CompilationTo run these files, you'll need a C++ compiler like g++ or clang.Bash# Example for Selection Sort
